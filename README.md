# Pacman Game

A Python Pacman game built with Turtle Graphics.

## Features
- Pacman controlled by arrow keys
- 4 ghosts with improved AI
- Custom board layout

## Controls
| Key | Action |
|-----|--------|
| Left  | Move left |
| Right | Move right |
| Up    | Move up |
| Down  | Move down |
